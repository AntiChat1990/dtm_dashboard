# Animations Skill

Use Framer Motion for animations.

Rules:
- Duration: 0.2–0.4s
- Use ease-in-out
- Avoid heavy animations

Preferred animations:
- opacity (fade)
- transform (scale, translate)
- subtle hover effects

Page transitions:
- Use AnimatePresence
- Smooth mount/unmount

Do NOT:
- Overanimate
- Use distracting effects
